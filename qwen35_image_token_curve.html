结论公式（推荐）：先将原图输入经过 smart_resize 得到 H', W'，则图像视觉 token 近似为 T_pad = ceil(H'/16) * ceil(W'/16) / 4。 在 Qwen3.5 路径中，H',W' 通常是 32 的倍数，因此等价为 T_pad = H' * W' / 1024。最终加入视觉边界后为 T_total = T_pad + 2。

粗略近似（按原始输入像素）：若直接估算，可用 P_in = H×W，先做像素截断 P_in' = clip(P_in, 65536, 16777216)，再用 T_pad ≈ P_in' / 1024。
